# 🌐 Django Static Website

## 📌 Description
This project is a basic static website built using Django.  
It demonstrates how to serve static pages like Home, About, and Contact using Django.

---

## 🚀 Features
- 🏠 Home page
- ℹ️ About page
- 📞 Contact page
- 🎨 Static files (CSS, images)
- ⚙️ Basic Django setup

---

## 🛠️ Tech Stack
- Python
- Django
- HTML, CSS

---
